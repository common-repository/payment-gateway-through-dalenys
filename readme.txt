== Description ==
Dalenys is a platform for online transactions offering exceptional acceptance rates without compromising on fraud. Allow your customers to pay for your products securely through Dalenys gateway.

= Plugin for = 
* Online-shops
* Online-schools
* Booking platforms
* and other sites that need safe transactions and secure deals

= Supported payment methods =
Visa, Mastercard, Girocard, American Express, PostFinance Cars, Maestro, Postpay, PayPal, Apple Pay, Masterpass, Visa Checkout, and others.


== Installation ==
= How to get API keys = 
Fill out the form on the site and request the information you need  https://www.dalenys.com/contact-us/ 
Get a mail from Dalenys with API KEY credentials.
Use API Key ID, API Key, Identifier, and API Secret key to set up the plugin.
